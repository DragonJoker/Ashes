Parameters

[in]subpassThe index of the subpass into which this buffer will be executed.
[in]frameBufferThe frame buffer into which the command buffer will render.
[in]occlusionQueryEnableTells if the command buffer can be executed while an occlusion query is active on main command buffer.
[in]queryFlagsThe occlusion query flags that can be used by an active occlusion query on the main command buffer, when this buffer is executed.
[in]pipelineStatisticsTells which pipeline statistics can be counted by an active query on the main command buffer, when this buffer is executed.
Returns
