◆ aspectMask

ImageAspectFlags renderer::ClearAttachment::aspectMask

Selects the colour, depth and/or stencil aspects of the attachment to be cleared. aspectMask can include renderer::ImageAspectFlag::eColour for colour attachments, renderer::ImageAspectFlag::eDepth for depth/stencil attachments with a depth component, and renderer::ImageAspectFlag::eStencil for depth/stencil attachments with a stencil component. If the subpass’s depth/stencil attachment is renderer::AttachmentUnused, then the clear has no effect.

uint32_t renderer::ClearAttachment::colourAttachment

Only meaningful if renderer::ImageAspectFlag::eColour is set in aspectMask, in which case it is an index to the colourAttaches array in the renderer::Subpass constructor of the current subpass which selects the colour attachment to clear. If colourAttachment is renderer::AttachmentUnused then the clear has no effect.
