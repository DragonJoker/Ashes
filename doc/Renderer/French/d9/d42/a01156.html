MemoryPropertyFlags  requirements,
uint32_t &  typeIndex

Déduit le type de mémoire à allouer, en fonction des informations données.

Paramètres
[in]typeBitsIl y a 32 types de mémoire décrits dans VkPhysicalDeviceMemoryProperties. Ce paramètre est une combinaison bit à bit décrivant les propriétés dans lesquelles il faut chercher.
[in]requirementsLes pré-requis pour le type de mémoire.
[in]typeIndexReçoit l'index du type de mémoire déduit.
Renvoie
true si le type a pu être déduit, false sinon.
