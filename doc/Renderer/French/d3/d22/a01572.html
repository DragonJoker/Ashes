template<typename FlagType>
class renderer::FlagCombination< FlagType >

Classe template qui fournit une conversion implicite depuis un scoped enum vers un type entier de base.

Remarques
Permet de définir des indicateurs via des combinaisons binaires (&, |), dont les opérandes seront obligatoirement de même taille binaire.
Paramètres
FlagTypeLe type de scoped enum.
